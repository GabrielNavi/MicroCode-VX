# vx-dga-l-microbit

Paquete Debian para Vitalinux que proporciona un acceso directo a MakeCode Micro:bit.

## Descripción

Este paquete instala un lanzador que permite acceder fácilmente a la plataforma web MakeCode Micro:bit desde el escritorio Vitalinux. MakeCode Micro:bit es un entorno integrado de desarrollo que permite a estudiantes de todas las edades aprender a programar usando una interfaz visual de bloques o código JavaScript/Python.

## Requisitos

- Navegador de Internet instalado, recomendable chromium.
- Conexión a Internet para acceder a la plataforma
